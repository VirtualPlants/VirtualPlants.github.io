Glossary

.. glossary::

    entry_point category
        blabla

    icon size
        image that describe well a concept, action, data, ... when image is small.
        Small means maximum 128x128 pixels. In other words, this image must be synthetic.

    preview size
        image that describe well a concept, action, data, ... when image has medium size.
        Medium size means 100 to 300 pixels. In other words, this image can be quite detailed.
