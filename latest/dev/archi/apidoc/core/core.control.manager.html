update(dic)[source]¶

Update controls with dict values. If a name is not referenced in ControlContainer, nothing is done.

Note

Example:

Let "container" a Container with one control "a" with value 1:

container:
    - a:IInt = 1

after container.update({'a':2, 'b':3}) we get

container:
  • a:IInt = 2
